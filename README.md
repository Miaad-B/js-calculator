# Simple JavaScript Calculator

This project is a **basic calculator** built as part of a JavaScript learning course.  
It performs simple arithmetic operations and uses **localStorage** to remember the last calculation when the page is reopened.

## Features

- Add, subtract, multiply, and divide
- Clear/reset functionality
- Displays the previous calculation using localStorage
- Simple and clean UI
- Built with vanilla JavaScript (no frameworks)

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM manipulation + localStorage)

## Learning Focus

This project helped me practice:

- Handling user input
- Updating the DOM dynamically
- Writing clean and reusable JavaScript functions
- Using `localStorage` to persist data
- Structuring a small JS project

## How to Use

1. Enter numbers using the buttons.
2. Choose an operation.
3. Press "=" to calculate.
4. The last result is saved and shown automatically when the page loads.
